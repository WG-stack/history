# 数显量表程序框架说明

## 📁 项目结构

```
USER_CODE/
├── app_main.h/c              # 应用主程序（状态机、主循环）
├── sensor_spi.h/c            # SPI传感器数据接收（EXTI+SPI+DMA）
├── key_scan.h/c              # 6键矩阵扫描
├── measurement.h/c           # 测量逻辑（ABS/REL/MAX/MIN/TIR/TOL）
├── settings.h/c              # 设置管理（用户设置+工程配置）
├── power_mgmt.h/c            # 电源管理（STOP模式、自动关机）
├── lcd_display.h/c           # LCD段码屏显示驱动
├── battery_monitor.h/c       # 电池电压监测
├── usb_output.h/c            # USB数据输出
├── uart_bluetooth.h/c        # 蓝牙串口通信
└── README.md                 # 本文档
```

## 🔧 模块说明

### 1. app_main - 应用主程序
**功能**：
- 状态机管理（开机/自检/正常/设置/错误）
- 主循环调度
- 按键处理
- 显示更新
- 错误检测

**状态**：
- `STATE_POWER_OFF`: 待机
- `STATE_SELF_TEST`: 自检（LCD全显、传感器检测）
- `STATE_NORMAL`: 正常测量
- `STATE_SETTING_MENU`: 用户设置菜单
- `STATE_ENGINEERING_MENU`: 工程配置菜单
- `STATE_TOL_SETTING`: 公差设置
- `STATE_ERROR`: 错误状态（Err0/1/2）

### 2. sensor_spi - 传感器接收
**功能**：
- PC12 EXTI下降沿触发
- 切换为SPI3_SCK接收3字节数据
- DMA自动接收
- BCD解码（24位数据帧）
- 单位识别（mm/inch）

**数据格式**：
```
[nibble0][nibble1][nibble2][nibble3][nibble4][控制位]
  ↓        ↓        ↓        ↓        ↓        ↓
  万位     千位     百位     十位     个位    符号+单位
```

### 3. key_scan - 按键扫描
**6键矩阵**：
```
PC13, PC14, PC15 组成3x2矩阵
- ON/ZERO: 开关机/清零
- ↓/ABS/SET: 相对绝对值/设置
- ↑/IN/MM: 英制公制/设置增
- TOL: 公差设置
- M: 分辨率/MAX/MIN/TIR
- ABS_ZERO: 原位开关
```

**事件**：
- 短按 (<1s)
- 长按 (≥3s)

### 4. measurement - 测量逻辑
**模式**：
- `MODE_ABS`: 绝对测量
- `MODE_REL`: 相对测量
- `MODE_MAX`: 最大值记录
- `MODE_MIN`: 最小值记录
- `MODE_TIR`: 跳动值 (MAX-MIN)

**分辨率**：
- 0.0005mm (万分表)
- 0.001mm (千分表)
- 0.01mm (百分表)

**指针显示**：
- ps-15 ~ ps+15
- ps-- (超-15)
- ps++ (超+15)
- ps0 (常显)

### 5. settings - 设置管理
**用户设置**：
1. 移动方向（向上/向下为正）
2. 自动关机时间（15min/30min/1H/2H/关闭）
3. 数据输出方式（USB/无线/无）
4. 显示倍数（正常/×2/÷2/自定义）

**工程设置**：
1. 传感器类型（0.0005mm/0.001mm）
2. 平均次数（0/4/8/16）
3. 传感器时钟（750kHz~2MHz）

### 6. power_mgmt - 电源管理
**功能**：
- 传感器电源控制（S_POWER: PC9）
- 传感器清零脉冲（S_ZERO: PB2, 50ms）
- STOP模式省电
- 自动关机
- USB检测（ADC PA4）
- 电池电压监测

**时序**：
- 上电：S_POWER高 → 延迟10ms → 启动时钟
- 下电：停止时钟 → 延迟10ms → S_POWER低

### 7. lcd_display - LCD显示
**段码表**：
- 4 COM × 26 SEG
- 6位数字显示
- 60个符号定义
- ps0~ps31指针显示

**显示内容**：
- 数值（带小数点）
- 单位（mm/inch）
- 模式（ABS/REL/MAX/MIN/TIR）
- 公差状态（OK/>|/|</TOL）
- 电池电量
- 指针

## 🔄 程序流程

### 开机流程
```
1. 检测开机键（长按ON/ZERO）
2. LCD全显2秒（自检）
3. 启动传感器电源和时钟
4. 等待数据（2秒超时）
5. 检查原位信号和数据稳定性
6. 自动置零（如果满足条件）
7. 进入正常测量模式
```

### 主循环
```
while(1) {
  1. 处理传感器数据
  2. 处理按键事件
  3. 更新测量数据
  4. 更新LCD显示
  5. 检查错误
  6. 更新自动关机
  7. 进入STOP模式省电
  8. 定时唤醒（按数据帧间隔）
}
```

### 错误处理
- **Err0**: 2秒无数据
- **Err1**: 无原位信号
- **Err2**: 数据跳动过大
